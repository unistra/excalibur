Installation¶

Installation from sources¶

To install Excalibur, you should get the app from github :

$> wget https://github.com/unistra/excalibur/archive/master.zip
$> unzip master.zip

Then install it :

$> cd excalibur-master
$> python setup.py install

Dependencies are automatically resolved and installed.
